# Petstore API Automation (Karate BDD)

## 📝 Description / Açıklama
**EN:** This project contains automated BDD (Behavior Driven Development) API tests for the Petstore service using the **Karate DSL** framework. It focuses on readability and maintainability of API test scenarios.
**TR:** Bu proje, Petstore servisi için **Karate DSL** framework'ü kullanılarak hazırlanmış otomatik BDD API testlerini içerir. API test senaryolarının okunabilirliği ve sürdürülebilirliği üzerine kurgulanmıştır.

---

## 🚀 Tech Stack / Teknolojiler
* **Java 17** & **Maven**
* **Karate Framework** (API Test Automation)
* **JUnit 5** (Test Runner)
* **Logback** (Logging)

---

**TR:** Testleri çalıştırmak için Maven'ın yüklü olması gerekmektedir. Şu komutu kullanın:
